Capabilities

Everything you need to ship faster

One harness. Four tools. Zero context switching between your code and your agent.

📖

Read any file

Pi reads your source code, configs, images — supports text files and screenshots. Truncated output for large files with smart offset navigation.

⚡

Run commands

Execute shell commands directly — ls, grep, find, npm, git. Timeout-safe with output streaming. No more copy-paste between windows.

✏️

Edit with precision

Targeted text replacements with exact matching. Multiple edits in one call. No diff fuzz. No accidental overwrites. Surgical code changes.

📝

Write new files

Create or overwrite files instantly. Auto-creates parent directories. Perfect for scaffolding, configs, and new components.

🧩

Extensible with skills

Custom skills, themes, extensions, and SDK integrations. Build your own tools on top of Pi. Tweak the TUI, keybindings, and prompts.

🔒

Runs where you work

Native terminal application. Works locally. Your code never leaves your machine unless you choose to. Full control over providers and models.

Tooling

Built for real engineering work

Pi understands your entire project — not just one file at a time.

✓

Reads images and screenshots

Share screenshots of designs, errors, or diagrams. Pi processes them as context.

✓

Smart file truncation

Large files auto-truncated at 2000 lines. Navigate with offset/limit. Never hit token limits blind.

✓

Batched edits

Multiple disjoint changes in a single file — one call. No overlapping edits. Merge nearby changes automatically.

✓

Timeout-safe commands

Long-running commands protected. Optional timeouts. Truncated output saved to temp files for review.

✓

Conventional Commits

Ultra-compressed commit messages with caveman mode. Subject ≤50 chars. Body only when "why" isn't obvious.

✓

Custom providers & models

Bring your own API keys. Swap models. Add custom providers. Full flexibility over your AI backend.
